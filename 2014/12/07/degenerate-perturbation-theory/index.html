second order degenerate perturbation theory

a summary of degenerate perturbation theory of second order

Question raised

We want to solve a Hamiltonian \(H=H_0+\lambda V\) perturbatively . What we know is the solution of \(H_0\), whose eigenstates are \(|n^0\rangle\) with eigenenergies \(E_n^0\) has no degeneracy; it also has a degenerate state \(|d_i^0\rangle\) with eigenenergy \(E_d^0\), where \(i=1,2,3\cdots\)g, the degenerate vector space \(D\) is thus spanned by \(|d_i^0\rangle\)

Solution

In general the degeneracy is lifted when we turn on the perturbation. \[ H|d_i\rangle=E_{di}|d_i\rangle \] However, we must keep in mind that when we switch off the perturbation, \(d_i\) is not necessarily tend to \(d_i^0\) as \(\lambda\to 0\). Rather, it will tend to a state \(|l_i^0\rangle\) which is a linear combination of \(|d_i^0\rangle\). Clearly, the vector space spanned by \(|l_i^0\rangle\) is also \(D\).

\[ |d_i\rangle=|l_i^0\rangle+\lambda|d_i^1\rangle+\lambda ^2|d_i^2\rangle+\cdots\\ E_{di}=E_d^0+\lambda E_d^1+\lambda^2E_d^2+\cdots \]

Substitute to the Shordinger eqution, we get:

First order:

\[ (H_0-E_d^0)|d_i^1\rangle=(E_d^1-V)|l_i^0\rangle \]

Using \(\langle d_j^0|\) to inner product, we get \(\langle d_j^0|(E_d^1-V)|l_i^0\rangle=0\), i.e. vector \((E_d^1-V)|l_i^0\rangle\) has no component in the vector space \(D\). We can then safely write :

\[\tag{1} \begin{aligned} |d_i^1\rangle&=\frac{1}{H_0-E_d^0}(E_d^1-V)|l_i^0\rangle\\ &=P\frac{1}{H_0-E_d^0}P(E_d^1-V)|l_i^0\rangle\\ &=-P\frac{1}{H_0-E_d^0}PV|l_i^0\rangle \end{aligned} \]

where \(P=\sum_{n\notin D}|n^0\rangle\langle n^0|\) is the projection operator. Clearly, \(|d_i^1\rangle\) also has no component in vector space \(D\).

Also, let \(|l_i^0\rangle=\sum_j c_{ij}|d_i^0\rangle\), we get the secular equation: \((E_d^1\delta_{ij}-V_{ij})c_{ij}=0\), solving this, we can get \(E_d^1\) and \(c_{ij}\) thus the vector \(|l_i^0\rangle\) that diagnolize \(V\).

Second order

\[ H_0|d_i^2\rangle+V|d_i^1\rangle=E_d^0|d_i^2\rangle+E_d^1|d_i^1\rangle+E_d^2|l_i^0\rangle \]

Using bra \(\langle l_i^0|\) to do the inner product, and provided \(|l_i^0\rangle\) is normalized, we get:

\[\tag{2} \begin{aligned} E_{di}^2&=\langle l_i^0|V|d_i^1\rangle \\ &=-\langle l_i^0|VP\frac{1}{H_0-E_d^0}PV|l_i^0\rangle \\ &=-\sum_{n\notin D}\frac{|\langle l_i^0|V|n^0\rangle|^2}{E_n^0-E_d^0} \end{aligned} \]

A very important remark

Note that in eqn(2) \(|l_i^0\rangle\) is the state that \(| d_i(\lambda)\rangle\) evolve to when we turn off the perturbation gradually (\(\lambda\to 0)\). As seen in first order perturbation , it is indeed the vectors that diagonalize \(V\), and the corresponding \(V_{ii}\) is just the first order energy.

However, when the degeneracy is not lifted in the first order. The “good” ket \(|l_i^0\rangle\) does not seems so good, because \(|d_i(\lambda)\rangle\) will still have ambiguity which final state to evolve as \(\lambda\to 0\).

In this case, we need second order to lift the degeneracy, and \(\Lambda=VP\frac{1}{H_0-E_d^0}PV\) plays the role of \(V\) as in the first order. the states that diagonalize \(\Lambda\) should be considered as “good”, you can construct a matrix of \(g\times g\) dimension using \(|d_i^0\rangle\) and obtain the good states \(|k_i^0\rangle\) as a linear combination of them, the corresponding \(\Lambda_{ii}=\langle k_i^0|\Lambda|k_i^0\rangle\) is thus the second order perturbation energy.
